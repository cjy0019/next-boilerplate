# NextJS boilerplate

**NextJS-boilerplate-Settings.**

### 사용된 기술들은 다음과 같습니다.

- react
- react-dom
- next
- typescript
- axios
- font(spoqa-Han-Sans-Neo-Bold)
- eslint(airbnb)
- prettier
- styled-reset
- styled-components
- redux, redux-toolkit
- redux-saga
- yarn

## 사용방법

```bash
git clone https://github.com/cjy0019/next-boilerplate.git

yarn install --immutable --immutable-cache --check-cache
```
